# 📊 Business Sales Dashboard

## 📌 Project Overview

A client-ready sales analytics dashboard built for **Data Science & Analytics **.  
Analyze revenue trends, top products, category/region performance, and get actionable business recommendations.


## 🎯 Objectives

* To track monthly sales trends
* To identify top-selling products
* To analyze sales across different countries

## 📊 Key Insights

Based on the included dataset:

 **Technology leads revenue** 
 **West & East regions dominate** 
 **Corporate segment is high-value** 
 **Heavy discounts erode profit** 
 **Seasonal peaks in Q4**
## 🛠 Tools Used

- **Python** — Data processing and dashboard
- **Streamlit** — Interactive web dashboard
- **Plotly** — Charts and visualizations
- **Pandas** — Data cleaning and aggregation

## Dashboard Features

•KPI cards
•Revenue trend
•Category & region
•Top products
•Segment analysis
•Filters
•Business insights
